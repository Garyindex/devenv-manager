# Security Policy

DevEnv Manager works with local filesystem paths and developer environment metadata. Please report security issues privately before public disclosure.

## Reporting

If you find a security issue, open a private GitHub security advisory or contact the maintainer privately.

Please include:

- Affected version
- Operating system
- Steps to reproduce
- Expected and actual behavior
- Potential impact

## Safety Notes

- Review all cleanup paths before quarantining or deleting files.
- Quarantine is intended as a safer default than permanent deletion.
- Do not share exported scan reports publicly if they include private local paths.
